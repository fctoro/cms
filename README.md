# FC Toro CMS

Bienvenue sur la documentation du CMS Éditorial pour **FC Toro**.
Ce projet est un CMS personnalisé pour permettre de piloter l'éditorial, les stages, et le contenu du site de l'académie FC Toro.

## Description

Le CMS gère vos :
- **Articles & Joueurs**
- **Stages & Candidatures**
- **Configuration Site (Staff, Elite, Flag Day)**
- **Utilisateurs & Accès**

## Lancement du Projet

Assurez-vous d'avoir configuré vos clés de base de données dans `.env.local`.

```bash
npm install
npm run dev
```
